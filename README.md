# Антон & Регина — свадебное приглашение 💍

Одностраничный сайт-приглашение на свадьбу. 21 августа 2026 · Ресторан Renaissance · Ташкент.

Статика без сборки: HTML + CSS + vanilla JS, анимации на GSAP 3 (ScrollTrigger).

## Запуск локально

```bash
python3 -m http.server 8091
# открыть http://localhost:8091
```

## Структура

- `index.html` — вся страница (hero-арка, приглашение, машина, дата + календарь, обратный отсчёт, место проведения, финал)
- `style.css` — стили (палитра пригласительной открытки: крем / коралл / шалфей)
- `script.js` — GSAP-анимации: прелоадер, интро с блокировкой скролла, голуби, машина по скроллу, календарь, отсчёт, музыка
- `assets/audio/music.mp3` — фоновая музыка (кнопка появляется автоматически, если файл есть)
- `assets/img/couple.png` — иллюстрация пары
